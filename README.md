# Medium Scraper Pro

Scrape Medium articles via Freedium mirror and save as **Markdown** and **JSON**.  
This app uses Streamlit with animations and styled output.

## Features

- Animated rainbow title
- Beautiful markdown cards
- Download Markdown & JSON
- Gradient hover buttons
- Spinner while fetching

## How to Run

```bash
git clone https://github.com/<username>/medium-scraper.git
cd medium-scraper
pip install -r requirements.txt
streamlit run app.py
